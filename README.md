# Question Paper Generator
This Node.js application generates a question paper based on given criteria such as difficulty levels and total marks.

### Setup
1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/question-paper-generator.git
   cd question-paper-generator
   
**Install dependencies:**
npm install

**Usage**

1. To generate a question paper, modify the parameters in app.js:
2. totalMarks: Total marks for the question paper.
3. percentages: Object specifying the distribution of marks for different difficulty levels.
4. Run the application:
5. node app.js
6. The generated question paper will be displayed in the console.

**Files**
1. [question.js: Defines the Question class]
2. [questions.js: Stores sample questions.]
3. [distribution.js: Calculates the distribution of marks.]
4. [questionPaperGenerator.js: Generates the question paper based on criteria.]
5. [shuffle.js: Utility function to shuffle arrays.]
6. [app.js: Entry point to test the application.]
